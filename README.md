# Blogy Sitesi  
Sisteme giriş yapan yazarlar yazılarını paylaşabilir,yorumlarını,mesajlarını yönetebelir.Admin tarafında ise yazarlar yönetilir.
# Kullanılan Teknolojiler
* AspNet Core 6.0
* N Tier Architecture
* Repository Desing Pattern
* Indetity
* Entity Framework Core
* Rapid Api
* Google Chart
* Çoklu Dil Desteği
